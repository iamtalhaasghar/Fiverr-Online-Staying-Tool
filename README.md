# Any Website-Online-Staying-Tool
A very easy and simple to use tool to stay online on Any Website as long as your PC is on.


# How to Use
<ol>
  <li>First of all Take the screenshot of the reload button located on your browser.</li>
  <li>Rename that screenshot file to "reload.png" and place that in a folder where executable file of this tool is located.</li>
  <li>If there is already a file present with the name "reload.png" you can replace it without any danger.</li>
  <li>Make sure the name of the file is "reload.png". Be aware that the all letters are in lower case. Changing the case of letters might ran you into a problem.</li>
  
  
</ol>
