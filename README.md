# Any Website-Online-Staying-Tool (Beta Version)
A very easy and simple to use tool to stay online on Any Website as long as your PC is on.

# Path for reload.png
Take screenshot of your browsers reload button and place it in your home directory.
On windows it should be like:
```C:\Users\USER_NAME\reload.png```
On Linux it should be like:
```/home/USER_NAME/reload.png```

You can see a ```reload.png``` file in this repository to see what it should look like.

# How to install
You can install this tool using pip using following command.
``` python
pip install tabreloader --upgrade
```

# How to run
# On Windows
Type ```python -m tabreloader``` in your console.
# On Linux
Typing just ```tabreloader``` should work but if not then try the full command as windows users will do.
